# Python Interview Topics for Security Engineers

- Log Parsing & Detection Rules. Skills: Dictionaries, datetime, and sliding windows
- Suspicious Process Detection. Skills: regex, string parsing, and pattern matching
- IP Frequency Analysis. Skills: collections and sorting
- Detect Impossible Travel. Skills: Dictionary grouping, timestamps, and logic building
- Suspicious File Extension Detection. Skills: string splitting and pattern detection
- Base64 Malware Detection. Skills: base64 module and regex
- Parse JSON Security Alerts. Skills: dictionary aggregation
- Detect Suspicious Domains. Skills: string comparison and Levenshtein logic
- Extract IOC from logs. Skills: regex and parsing unstructured data
- Build a Simple Threat Detection Engine. Skills: dictionaries and rule engine logic
- Sliding Window Detection (SOC Use Case). Skills: timestamps and queue/window logic
- Hash Reputation Check. Skills: sets and lookup efficiency

## Common interview skills and libraries

- dictionaries, lists, sets, loops, functions, conditionals, operators, and comprehensions
- re, collections, datetime, json, and base64
